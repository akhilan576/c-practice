# c-practice 💻

A collection of beginner-level programming exercises written in C — focused on building a strong foundation in low-level programming concepts.

## 📋 Project Overview

This repository serves as a personal practice space for learning the C programming language from scratch. Each file explores a core concept in C through hands-on coding exercises.

## 🧠 Topics Covered

- Variables, data types, and operators
- Control flow: if/else, switch statements
- Loops: for, while, do-while
- Functions and recursion
- Arrays and strings
- Pointers and memory management
- Structs and file I/O (coming soon)

## 🛠️ Tech Stack

- **Language:** C
- **Compiler:** GCC
- **IDE:** VS Code / any C-compatible editor

## 🧑‍💻 How to Run

1. Clone the repository:
   git clone https://github.com/akhilan576/c-practice.git

2. Navigate to the directory:
   cd c-practice

3. Compile and run a file:
   gcc filename.c -o output
   ./output
